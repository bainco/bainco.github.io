The big idea behind this project is to emphasize an important set of practices in programming: the ability to read, understand, and critique existing computer code. We currently do this informally through ungraded activities, however, in order to better emphasize the fact that these skills are a key learning goal in the course, I would like to integrate graded assessments in the course focused on these ideas and practices.

> Implementing Peer Review as a Course Learning Objective in CS 110

## Narrativizing
This will be an activity where students are given a program and asked to add comments and/or rewrite the code to it in order to make it more transparent (i.e. readable as if it were written in plain English).

## Critiquing
Following Google’s Code Review Practices, students will be given a program and asked to critique it across multiple dimensions: Design, Functionality, Transparency, Comments, and Style. Each of these will be defined as part of the course’s lecture component with appropriate examples

## Deliverables
* New Pre-Course Survey for CS 110
* Short Multiple Choice Activity on Narrativizing Computer Programs
* Short Multiple Choice Activity on Critiquing Computer Programs
* Students complete a Narrativizing activity by reviewing a single function from another student’s project
* Students complete a Peer Review activity by reviewing another team’s Project 2 Submission
* Summarize Experiences in a report (target format: 4 page SIGCSE conference paper)

## Remaining Project Issues
* I would like this to be a design-based research project. This would require planning out the assignments such that, in addition to student responses, there might be reflection questions (both quantitative and qualitative)
* I’m not sure how to integrate this into my existing syllabus assignment categories (Quizzes, Mini-Quizzes, Tutorials, Homeworks, Projects)
* For deliverable 3 and 4, I actually have to build something to allow this to work in Canvas as it’s not currently possible. I can think of some temporary ways to do it, but I’d like to have a more robust platform to use in other classes as well
* To make the class more unified, I’m considering adding these sorts of questions to the exams as well but am worried about grading those responses equitably.
